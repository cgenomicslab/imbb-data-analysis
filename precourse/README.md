# Python & R Crash Course

A brief introduction to Python and R for people with no prior programming experience. Go through these notebooks before the course to familiarize yourself with the basics. Download them and clear all cell outputs to work through them step-by-step.

## Notebooks

| # | Topic | Content |
|---|-------|---------|
| 0 | [Jupyter & Python Basics](notebooks/00_jupyter_and_python_basics.ipynb) | Jupyter environment, keyboard shortcuts, variables, data types, arithmetic, strings |
| 1 | [Lists, Dictionaries & Control Flow](notebooks/01_collections_and_control_flow.ipynb) | Lists, dictionaries, indexing, slicing, if/else, for loops |
| 2 | [Functions, Imports, NumPy & Pandas](notebooks/02_functions_imports_numpy_pandas.ipynb) | Defining functions, importing modules, NumPy arrays, pandas DataFrames |
| 3 | [Introduction to R](notebooks/03_introduction_to_R.ipynb) | R basics, vectors, data frames, functions, packages, Python-vs-R reference table |

Notebooks 0–2 cover Python (used in Week 1). Notebook 3 covers R (used in some Week 2 sessions).

Each notebook contains short "Try it" cells where you can test what you just learned.

## Requirements

**Python:** 3.11 with `numpy` and `pandas`. See the [setup guide](../docs/SETUP.md) for installation instructions.

**R:** The R notebook runs in Jupyter via the [IRkernel](https://irkernel.github.io/). If you are using the course JupyterHub, this is already set up. The Week 2 instructors will provide specific package installation instructions.
